### 백준 no.10951 A+B -4

- 문제: 정수 A, B를 입력받은 다음, A+B를 출력하는 프로그램을 작성

- 특이사항: 여러 개의 테스트 케이스로 이루어져있으나 별도 테스트 케이스 지정 없음

- 통과 코드
  
  ```python
  while input:
      try:
          A, B = map(int, input().split())
          print(A+B)
      except:
          break
  ```

- 매번 테스트 케이스가 주어지는 문제들만 봐와서 어떻게 해결하는지 생각 안나는 바람에 결국 구글링해서 알아냄. try-except를 이렇게도 사용할 수 있다는 것 기억하기.
