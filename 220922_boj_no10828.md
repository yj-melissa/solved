### 백준 10828 스택

- 1차 시도 코드
```python
N = int(input())  
stack = [0]*10000  
top = -1  
for _ in range(N):  
    order = list(input().split())        # push는 입력으로 숫자 같이 들어오므로  
  
    if order[0] == 'push':  
        top += 1  
        stack[top] = order[1]  
  
    elif order[0] == 'pop':  
        if top == -1 :  
            print(-1)  
        else:  
            top -= 1  
            print(stack[top+1])  
  
    elif order[0] == 'size':  
        if top <= -1:  
            print(0)  
        else:  
            print(top+1)  
  
    elif order[0] == 'empty':  
        if top <= -1:  
            print(1)  
        else:  
            print(0)  
  
    elif order[0] == 'top':  
        if top <= -1:  
            print(-1)  
        else:  
            print(stack[top])
```
