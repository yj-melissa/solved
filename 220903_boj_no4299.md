### 백준 no.4299 AFC 윔블던

- 문제: 두 숫자의 합과 차가 주어졌을 때, 두 숫자를 구하는 프로그램 작성.

- 입력: 숫자는 항상 음이 아닌 정수.

- 출력: 더 높은 숫자를 먼저 출력, 만약 그러한 합과 차를 갖는 경우가 없다면 -1을 출력



- 실패 코드
  
  ```python
  plus, minus = map(int, input().split())
  # a + b = plus
  # a - b = minus
  # a = (plus + minus)
  # b = (plus - a)
  a = (plus + minus) // 2
  b = plus - a
  if b != a - minus:
      print(-1)
  else:
      if b > a:
          a, b = b, a
      print(a, b)
  
  ```
